Overall Design

I think I had done an alright job when it comes to design. I used a really basic layout and incorporated white space and a minimalistic colour scheme. I have kept the design consistent across all pages and tried to add some personality where I feel it best fits. With more time, this page could be a lot cooler but as for an assignment I got done in a week, the design is pretty neat.


Technical Aspects

The site uses HTML and CSS alone along with Google fonts. In terms of the CSS selectors, I‚Äôve used them to help style the HTML elements and make them conform to the design I want. The selectors were just Simple Selectors, using the name, id and class. In terms of aesthetics, I was going for a design very much similar to my portfolio. I thought it was simple and had a design that was well received (minimalistic and fun). I included the use of emojis to add some personality to the page and make it more relatable. People, myself included love emojis apparently.

The image used in the header container is rounded since it‚Äôs a personal preference of mine to have rounded images where ever possible. The background colour is an offwhite since a stark white background may have not been easy on the eyes. An offwhite background is very common among professionally designed webpages since it does give the page some depth and contrast.
